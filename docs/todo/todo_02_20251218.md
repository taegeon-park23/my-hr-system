## 3. 프론트엔드 작업 우선순위 제안
1.  **[Critical] Approval Action UI**: 결재 승인/반려 버튼 및 API 연동 (기본 기능 작동 필수).
2.  **[High] Dashboard Expansion**: 팀장급 이상을 위한 통계/캘린더 위젯 추가.
3.  **[Medium] Global Admin Console**: 슈퍼 어드민용 별도 레이아웃 및 테넌트 CRUD 페이지 신설.
4.  **[Medium] Approval Timeline**: 결재 진행 상황 시각화 컴포넌트 개발.
