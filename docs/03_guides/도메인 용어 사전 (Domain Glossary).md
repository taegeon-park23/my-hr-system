# **도메인 용어 사전 (Domain Glossary)**

## **1\. 개요**

본 문서는 HR 시스템 프로젝트에서 사용하는 용어를 정의한다. 기획자, 개발자, 디자이너는 본 문서에 정의된 \*\*유비쿼터스 언어(Ubiquitous Language)\*\*를 사용하여 소통해야 하며, 코드 내의 클래스명, 변수명, DB 컬럼명도 이를 따른다.

## **2\. 일반 및 테넌트 (General & Tenant)**

| 한글 용어 | 영어 용어 (Code) | 설명 | 비고 |
| :---- | :---- | :---- | :---- |
| **테넌트 / 고객사** | Tenant / Company | 시스템을 사용하는 회사 단위. 데이터 격리의 최상위 기준. | company\_id |
| **시스템 관리자** | Super Admin | 전체 테넌트를 관리하는 플랫폼 운영자. |  |
| **테넌트 관리자** | Tenant Admin | 특정 고객사의 최고 관리자. |  |
| **사용자** | User | 시스템에 로그인할 수 있는 계정 주체. | 인증(AuthN) 관점 |
| **임직원** | Employee | 회사에 소속되어 일하는 사람. 인사 정보의 주체. | HR 관점 |

## **3\. 조직 및 인사 (Organization & HR)**

| 한글 용어 | 영어 용어 (Code) | 설명 | 비고 |
| :---- | :---- | :---- | :---- |
| **부서 / 조직** | Department (Dept) | 회사의 하위 조직 단위. 트리 구조를 가짐. | Organization은 모듈명으로만 사용 |
| **부서장** | Dept Leader / Manager | 해당 부서를 대표하고 관리 권한을 가진 사람. |  |
| **직위** | Position | 서열상의 등급 (예: 사원, 대리, 과장, 부장). | 인사체계 |
| **직책** | Job Title / Duty | 맡은 보직 (예: 팀장, 본부장, 실장). | 조직체계 |
| **직무** | Job Function | 수행하는 업무의 종류 (예: 개발, 디자인, 영업). |  |

## **4\. 결재 (Approval)**

| 한글 용어 | 영어 용어 (Code) | 설명 | 비고 |
| :---- | :---- | :---- | :---- |
| **결재 요청 / 기안** | Approval Request | 승인을 받기 위해 문서를 작성하고 제출하는 행위. |  |
| **기안자** | Requester | 결재를 올린 사람. |  |
| **결재자** | Assignee / Approver | 해당 단계에서 승인/반려 권한을 가진 사람. |  |
| **참조자** | Referrer | 승인 권한은 없으나 진행 상황과 결과를 볼 수 있는 사람. |  |
| **결재선** | Approval Line | 결재가 거쳐가는 경로와 순서. |  |
| **단계** | Step | 결재선의 각 순서 (예: 1차 검토, 2차 승인). |  |
| **합의** | Consensus | 하나의 단계에서 여러 명이 승인해야 통과되는 방식. |  |
| **대결 (대리결재)** | Delegation | 결재자가 부재 시 지정된 대리인이 대신 승인하는 것. |  |

## **5\. 정책 및 권한 (Policy & Auth)**

| 한글 용어 | 영어 용어 (Code) | 설명 | 비고 |
| :---- | :---- | :---- | :---- |
| **권한 / 인가** | Authorization (AuthZ) | 특정 리소스나 기능에 접근할 수 있는 자격. |  |
| **데이터 범위** | Data Scope | 사용자가 조회할 수 있는 데이터의 영역 (본인, 부서, 전체). |  |
| **접근권 / 티켓** | Access Grant (Ticket) | 특정 조건(결재 승인 등) 만족 시 부여되는 임시 권한. | 동적 권한 |
| **역할** | Role | 권한의 집합 (예: USER, MANAGER, ADMIN). | 정적 권한 |

## **6\. 근태 및 급여 (Attendance & Payroll)**

| 한글 용어 | 영어 용어 (Code) | 설명 | 비고 |
| :---- | :---- | :---- | :---- |
| **근무 유형** | Work Schedule | 시차출퇴근, 고정근무 등 근무 형태. |  |
| **출근 / 퇴근** | Check-in / Check-out | 업무 시작과 종료 시각을 기록하는 행위. |  |
| **급여 대장** | Payroll | 월별 전 직원의 급여 지급 내역 원장. |  |
| **급여 명세서** | Payslip | 개인에게 발급되는 상세 급여 내역서. |  |
| **연차 / 휴가** | Vacation / Leave | 유급 휴가. (시스템 내에서는 Vacation으로 통일 권장) | Annual Leave |
| **잔여 연차** | Remaining Days / Balance | 사용할 수 있는 남은 휴가 일수. |  |
